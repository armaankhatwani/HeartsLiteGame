PROJECT TITLE: Hearts Lite
PURPOSE OF PROJECT: Created a multi-lab project using java in which there is a smart, computer-run game of Hearts Lite (simplified version of the game Hearts). Multiple ArrayList data structures and methods used alongside inheritance and polymorphism. 
VERSION or DATE: May 24, 2023
HOW TO START THIS PROJECT: Download the folder, run BlueJ app, and use the TestCardPlayerLevel1 to see the computer play the game of Hearts Lite
